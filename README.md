UppSkill MVC

Plataforma web desarrollada con arquitectura MVC que conecta mentores y aprendices, permitiendo el registro de usuarios y la gestión de roles dentro del sistema.

### Tecnologías utilizadas
PHP 8
Arquitectura MVC
MySQL
Sass (SCSS)
Gulp
JavaScript

### Instalación
Clona el repositorio:
git clone https://github.com/tuusuario/UppSkill-MVC.git
Instala dependencias:
composer install
npm install
Compila los estilos:
npm run dev
Inicia el servidor local:
php -S localhost:3000 -t public
Abre en el navegador:
http://localhost:3000
